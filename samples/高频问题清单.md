## Agent Coding Checklist

Check these when generating code:

### Security
- [ ] No hardcoded secrets/passwords
- [ ] No plaintext hashes in SQL scripts
- [ ] No SQL injection risks

### Java / Spring Boot
- [ ] `application.yml` uses `${}` placeholders
- [ ] No excessive string literal duplication

### Shell Scripts
- [ ] Use `[[ ]]` for conditionals
- [ ] Positional params assigned to local vars

### Frontend
- [ ] Text contrast meets WCAG AA
- [ ] `<label>` associated with input
- [ ] No nested ternary expressions
